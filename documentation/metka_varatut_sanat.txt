Lista Metka:n varatuista sanoista joita ei saa käyttää kenttäavaimina:
- key
- id
- no
- configuration
- type
- version
- state
- removed
- time
- user
- approved
- draft
- handler
- saved
- value
- general

Näiden kenttäavaimien välttäminen vähentää virhetilanteiden mahdollisuutta varsinkin hakujen yhteydessä. Kenttäavain kattaa kaiken mitä hakua kirjoittaessa erotetaan pisteellä. Näitä termejä voi käyttää osana muita termejä, esimerkiksi "dataversion" ei aiheuta mitään ongelmia. Varattuja sanoja käytetään usein vain viimeisenä tai ensimmäisenä terminä kenttäavaimessa, mutta ongelmien välttämiseksi kannattaa pyrkiä välttämään näiden termien käyttöä datan kuvailuissa.
